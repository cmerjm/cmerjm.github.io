# Responsive Static Website

## *Mémo CLI github:*

##### Cloner le projet depuis le repository distant, sur mon ordinateur local:

```cli
git clone https://github.com/cmerjm/cmerjm.github.io.git
```

##### Récupérer des mises à jour sur le repository(pull):

*si un commit est plus récent que le mien je vais devoir le récupérer avant d'envoyer le mien*

```cli
git pull
```
---

#### *Faire des modifications dans mon code.*

###### Exemple: 

> Je viens de modifier le fichier *index.html* situé sur mon ordinateur local, pour l'envoyer à mon repository distant(github), je vais suivre les trois étapes suivantes:

##### 1) Je prépare mon colis(staging area):

*je viens ajouter dans mon colis mon élément modifier(index.html)*

```cli
git add index.html
```

##### 2) J'écris sur mon colis ce qu'il contient(commit):

*J'essaie d'être le plus clair possible tout en restant simple*

```cli
git commit -m "Modification Navbar"
```

##### 3) J'envoie mon colis !(push):

*Maintenant mon colis est prêt pour envoi !*

```cli
git push
```
